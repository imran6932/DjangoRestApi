# DjangoRestApi

__Django Rest API with Signup and forgot password__

***Third party packages are used***

 * **JWT**

 * **dj-rest-auth**

 * **allauth**
 
 # Features

***PostgreSQL database***

* ***Signup***

* ***Login***

* ***Logout***

* ***Reset/Forgot Password***

* ***3 User Level Super-Admin, Teacher and Student***
